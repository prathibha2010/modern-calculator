# 🧮 Modern Calculator Web Application

## 📌 Overview
The Modern Calculator is a responsive web-based application developed using HTML, CSS, and JavaScript. It performs both basic and advanced mathematical operations and provides a clean, interactive user interface with dark and light theme support.

---

## 🚀 Features

### 🔹 Basic Operations
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

### 🔹 Advanced Operations
- Square Root (√)
- Square (x²)
- Cube (x³)
- Percentage (%)
- Floor Division (//)
- Reciprocal (1/x)
- Backspace (⌫)
- Clear Display (C)

### 🔹 User Interface Features
- Dark / Light mode toggle 🌙🌞
- Gradient background design
- Interactive buttons with hover effects
- Real-time expression evaluation
- Responsive layout for different screen sizes

---

## 🛠️ Technologies Used
- HTML5 – Structure of the application  
- CSS3 – Styling, layout, and themes  
- JavaScript – Functionality and logic  

---

## 📂 Project Structure
Calculator/
│── index.html
│── style.css
│── script.js

## ⚠️ Limitations
- Uses `eval()` for expression evaluation (not recommended for production)  
- Limited to basic and intermediate mathematical operations  

---

## 🎯 Future Enhancements
- Add scientific calculator functions (sin, cos, log)  
- Keyboard input support  
- Calculation history tracking  
- Improved security without using eval()  
- UI enhancements with animations
